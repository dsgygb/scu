###1.江安空调如何定时？

微博网友@_huuuuugh：长摁M键，然后就会显示当前时间并闪烁，再短摁M键就可以摁上下键调节当前时间，然后再短摁M键，就能调节定时开和定时关的时间了，调节好定时关的时间后，按风速键，定时完毕。定时成功后，显示器上面会显示有定时关的字样
